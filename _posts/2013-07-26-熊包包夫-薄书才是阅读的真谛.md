那些薄薄一小册的书已经越来越少了，现在一本像样一点的书一般是在30元以上，9元~20元之间的书，真的是少之又少，而薄书的问题主要是成本过高，书号资源紧张而且需要用钱向大社买，总之，薄书是奢侈品，这也造成了一件好事，就是薄书一般都是有自信，有读者缘的好书。

薄书拿着舒服，一般都是平装，书轻，纸张也轻，可以躺着看，地铁上站着看，包里揣一本儿，三天的上下班就圆满看完，是件愉快的事。

说说我这几年买的基本印象深刻的薄书吧。

![](https://t2.qpic.cn/mblogpic/f033223d8859374dc6b6/2000)

《议事规则》 （美）亨利·M·罗伯特（著） / 商务印书馆 / 1995-11 / 18.00元

这本书非常好，简明扼要，每个开会的人都应该有一本，比如业主大会等等，能有效地解决人们讨论时间。学会开会，尊称规则，是有民主能力的开始。一百多年来，罗伯特议事规则已经被证实了是高效的。

![](https://t2.qpic.cn/mblogpic/3b795e04915babb710ae/2000)

《常识》 （美）托马斯·潘恩（著） / 何实（译） / 华夏出版社 / 2004-1 / 13.00元

这书不错，不过这么小的版本真的不好找了，现在的多数价格都超过了20元。这是独立战争前美国人除了圣经之外保有最多的书，要读懂美国的过去，这本书和《联邦党人文集》是一定要看的。

![](https://t2.qpic.cn/mblogpic/a22b7b24a630ac3964fe/2000)

《中国史学名著》 钱穆 / 生活·读书·新知三联书店 / 2005-02 / 22.00元

这书卖到了22元，我对不起大家，万幸有种东西叫做折扣。钱穆先生的这本书是历史爱好者的入门指导。

![](https://t2.qpic.cn/mblogpic/0de409d534b0a457db50/2000)

《诗词格律概要》 王力 / 北京出版社 / 2002-1 / 12.00元

和愚蠢的老干体和青涩的打油体告别的入门级读物，王力先生的这本薄书非常严谨，律、绝、古和词牌的格律都有了，如果需要装腔，这本书完爆那些不懂韵脚、平仄、粘和对的愚昧诗歌臭手。

![](https://t2.qpic.cn/mblogpic/932a65e71f7c29403806/2000)

《敦煌》 （日）井上靖（著） / 刘慕沙（译） / 北京十月文艺出版社 / 2010-10 / 20.00元

井上靖的这部作品当年曾经被拍成电影，中日合拍，80年代的时候多次出现在央视二套等节目上，一个背井离乡去西夏冒险被抓兵的宋朝读书人，在失去恋人之后是如何报仇、保护敦煌书卷的故事，文字原著来看，是个非常优美和忧伤的故事。

![](https://t2.qpic.cn/mblogpic/746178483de51d9cab1a/2000)

《论农业》 （古罗马） M·T·瓦罗（著） / 王家绶（译） / 商务印书馆 / 1981-6 / 15.00元

15元应该是再版的价格，1981年的这本书估计才几毛钱。这本书写的是古罗马时代的田园经营。今天你去法国、西班牙的庄园旅游或者买葡萄园之前，不妨读一读，然后在你的女友或者女儿面前大方地卖弄一下古罗马的知识吧，因为传统农业，其实没什么变化。

![](https://t2.qpic.cn/mblogpic/63ea95c002aae4fb8536/2000)

《长征记》 （希腊）色诺芬（著） / 崔金戎（译） / 商务印书馆 / 1985-5 / 9.50元

其实是讲一个打败仗的故事，当一次战败之后，一万雇佣希腊步兵是如何躲开充满敌意的敌人，组织起来突围，用最小的代价完成撤退的。作者色诺芬称自己是挺身而出临时担任指挥的，逆境中，一定要有承压重担的那个人。

《罗马十二帝王传》 （古罗马）苏维托尼乌斯（著） / 张竹明、王乃新、蒋平 等（译） / 商务印书馆 / 2000-02 / 18.00元

这本书可能是这套薄书记里最厚的一本，写的是罗马十二位君主的人物传记，这是写人物文章传记无法绕过的一本书。不因为人的暴行就把他写得一无是处，尽量复原一个人的一切，难得的好传记。

![](https://t2.qpic.cn/mblogpic/cce4c318b2fd117d7d38/2000)

《鸭子说不可以》（英）艾莉森·里奇 / 2013-1 / 12.80元

儿童绘本，但是非常值得咀嚼。池塘管理员大白鹅要出门度假，把管理权交给了鸭子，鸭子就开始管闲事，不许跳水、不许比赛飞行、不许比赛游泳。把池塘里的动物，青蛙啊蜻蜓啊都轰走了。结果鸭子面临着一个孤零零的池塘，最后他扔掉禁令，把大家从草地上都请了回来。鸭子的池塘成了世界上最自由的池塘。

这是一个歌颂自由市场的故事，管制重重干预过多的市场，人们会用脚投票。不要做鸭子式的管理员，才会有经济上的活力。
